# Skllentrix_intern
This project uses machine learning techniques such as Content-Based Filtering, Collaborative Filtering, and Hybrid Recommendation Systems. It calculates the similarity between movies using Cosine Similarity and recommends the most relevant movies to the user. The MovieLens Dataset is commonly used for training and testing the recommendation model.
Tools and Technologies
Programming Language: Python
IDE: Jupyter Notebook / VS Code / Google Colab
Libraries:
Pandas – Data manipulation
NumPy – Numerical computations
Scikit-learn – Machine learning algorithms
TfidfVectorizer – Text feature extraction
Cosine Similarity – Measure similarity between movies
Surprise Library – Collaborative filtering
Matplotlib & Seaborn – Data visualization
Dataset: MovieLens Dataset
